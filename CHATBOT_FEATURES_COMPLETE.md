## 🚀 Usage Guide

### Starting New Chat:
1. Click "New Chat" button
2. Type message
3. Chat auto-saves

### Searching Chats:
1. Use search bar in sidebar
2. Type keywords
3. Filtered results appear instantly

### Copying Messages:
1. Hover over any message
2. Click copy icon
3. Content copied to clipboard
4. Check mark confirmation

### Editing Messages:
1. Hover over your message
2. Click edit icon
3. Modify text
4. Click "Save" or "Cancel"

### Regenerating AI Response:
1. Hover over AI message
2. Click regenerate icon
3. New response generated
4. Replaces old response

### Pinning Messages:
1. Hover over message
2. Click pin icon
3. Yellow highlight appears
4. Pin again to unpin

### Deleting Messages:
1. Hover over message
2. Click trash icon
3. Confirm deletion
4. Message removed

### Exporting Chat:
1. Click "Export Chat" in sidebar
2. Download .txt file
3. Opens in text editor
4. Contains all messages

### Dark Mode:
1. Click "Settings" button
2. Toggle "Dark Mode" switch
3. Theme changes instantly
4. Preference saved

### Timestamps:
1. Open Settings
2. Toggle "Show Timestamps"
3. Times appear on messages
4. Format: 10:30 AM

## 🎯 Technical Details

### State Management:
- 15+ state variables
- localStorage persistence
- Auto-save on changes
- Dark mode preference
- Pinned messages array

### Features Breakdown:
```javascript
- messages: Chat history
- darkMode: Theme toggle
- searchQuery: Filter chats
- showSettings: Settings modal
- editingMessageIndex: Edit state
- copiedIndex: Copy confirmation
- pinnedMessages: Pinned array
- showTimestamps: Time display
```

### API Integration:
- apiService.chat() - Send messages
- apiService.getChatHistory() - Load chats
- apiService.saveChatHistory() - Save new
- apiService.updateChatHistory() - Update existing
- apiService.deleteChat() - Remove chat
